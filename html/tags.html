Tags Html.

  • Tags H: <h1>, <h2>, <h3>,<h4>, <h5>, <h6>

    Tags para definir cabecalhos(titulos), sendo que h1 e o titulo principal, h2 subtitulo e assim por diante, só deve existir um h1(titulo principal) por pagina

  • <a> Links

    Serve para criar links para uma pagina externa ou interna.

  • <p> Paragrafo

    Usada para definir paragrafos de texto.

  • <ul> Lista

    A tag <ul> marca uma lista de itens sem ordem (unordered list). Cada item da lista deve ser marcado com uma <li>.

    Acessibilidade: Listas são anunciadas pelo leitor de tela e o usuário é informado previamente quantos itens existem na lista.

  • <ol> Lista

    A tag <ol> marca uma lista de itens ordenada(ordered list). Cada item da lista deve ser marcado com uma <li>.

    Acessibilidade: Listas são anunciadas pelo leitor de tela e o usuário é informado previamente quantos itens existem na lista.

  • <img> Imagem

    Serve para inserir uma imagem na paǵina.

  • <div> Divisor

    Container genérico para conteúdo de fluxo, que de certa forma não representa nada. Ele pode ser utilizado para agrupar elementos para fins de estilos (usando class ou id).

  • <form> Formulario

    Representa uma seção de um documento que contém controles interativos que permitem ao usuário submeter informação a um determinado servidor web

  • <input type="password"> Input

    É usado para criar controles interativos para formulários baseados na web para receber dados do usuário. O tipo de input que será depende do seu atributo type, se nao for especificado por padarao sera text.

    Atributo type: Valores que podem ser usados

    • checkbox: Uma caixa de marcação
    • color: Um controle para especificar cores.
    • date: Um controle para inserir uma data (ano, mês e dia, sem horário)
    • email: Um campo para inserir um endereço de e-mail.
    • file: Um controle que permite ao usuário selecionar um arquivo.
    • hidden: Um controle que não é exibido mas cujo valor é enviado ao servidor.
    • image: Um botão gráfico para enviar o formulário. Você deve usar o atributo src para definir a fonte da imagem e o atributo alt para definir um texto alternativo.
    • number: Um controle para inserir um número de ponto flutuante.
    • password: Um campo de texto com uma só linha cujo valor é obscurecido. Use o atributo maxlength para especificar o comprimento máximo do valor que pode ser inserido.
    • radio: Um botão de escolha.

    Obs: Consulte a documentacao para verificar demais atributos desse elemento como: placeholder, required...

  • <button> Botão

    Representa um botão clicavel. Para saber quais atributos pode ser utilizadoo em botao como por exmplo disabled e type consulete a documentacao do Mdn.

Tags Semanticas.

  • <main>

    Serve para definir o conteúdo principal da página ou da aplicação

  • <header>

    Representa um grupo de suporte introdutório ou navegacional. Pode conter alguns elementos de cabeçalho mas também outros elementos como um logo, seções de cabeçalho, formulário de pesquisa, e outros.

  • <nav>: Navegação

    Representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.

  • <section>: Secao

    Representa uma parte do conteudo da sua pagina, como por exemplo, uma secao de produtos.

  • <header>: Rodape

    Define o rodapé de uma pagina oude uma secao
