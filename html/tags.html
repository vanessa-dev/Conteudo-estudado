Tags Html.

  • Tags H: <h1>, <h2>, <h3>,<h4>, <h5>, <h6>

    Tags para definir cabecalhos(titulos), sendo que h1 e o titulo principal, h2 subtitulo e assim por diante, só deve existir um h1(titulo principal) por pagina

  • <a> Links

    Serve para criar links para uma pagina externa ou interna.

  • <p> Paragrafo

    Usada para definir paragrafos de texto.

  • <ul> Lista

    A tag <ul> marca uma lista de itens sem ordem (unordered list). Cada item da lista deve ser marcado com uma <li>.

    Acessibilidade: Listas são anunciadas pelo leitor de tela e o usuário é informado previamente quantos itens existem na lista.

  • <ol> Lista

    A tag <ol> marca uma lista de itens ordenada(ordered list). Cada item da lista deve ser marcado com uma <li>.

    Acessibilidade: Listas são anunciadas pelo leitor de tela e o usuário é informado previamente quantos itens existem na lista.

Tags Semanticas.

  • <main>

    Serve para definir o conteúdo principal da página ou da aplicação

  • <header>

    Representa um grupo de suporte introdutório ou navegacional. Pode conter alguns elementos de cabeçalho mas também outros elementos como um logo, seções de cabeçalho, formulário de pesquisa, e outros.

  • <nav>: Navegação

    Representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.
