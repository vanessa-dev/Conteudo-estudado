Seletor Css e Pseudo Classes

Os estilos são aplicados em "cascata", um acima do outro. A ordem e especificidade do seletor importam no CSS.

  • Selecionar elemento por tag.

    h1 {color: #000;}

    Para selecionar um atributo por tags htmls e necessario somente passar o nome d euma tag html existente: h1

  • Selecionar elemento por id.

    <h1 id="color"> Teste </h1> #color{color: #000;}

    Atributo HTML que adiciona um identificador único na tag. Esse identificar pode ser utilizado no CSS para selecionarmos o elemento: #nomeid.

  • Selecionar elemento por classe.

    <h1 classe="color"> Teste </h1> .color{color: #000;}

    Atributo HTML que adiciona um identificador reutilizável na tag. Esse identificar pode ser utilizado no CSS para selecionarmos o(s) elemento(s): .classe

  • h1, p

    A vírgula permite selecionarmos múltiplos elementos para a aplicação de um mesmo estilo.

  • p a

    Seleciona o a que tiver um p como elemento pai (não precisa ser filho direto).

  • :first-child

    Seleciona o primeiro elemento.

  • :last-child

    Seleciona o último elemento.

  • :nth-child(4)

    4 (quarto elemento), even (pares), odd (ímpares), 3n (de 3 em 3).

  • :nth-child(4)

    4 (quarto elemento), even (pares), odd (ímpares), 3n (de 3 em 3).
